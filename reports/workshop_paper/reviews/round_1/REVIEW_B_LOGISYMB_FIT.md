# Round 1 Review B: LogiSymb Fit

## Must Fix

- Formalize what consistency means for role reversal and negation. Role reversal should be presented as predicate-argument permutation changing event truth conditions. Negation should be presented conservatively because the source data mix negation with predicate/category foils.
- Do not imply that shift-surprisal correlation alone is logical correctness.
- Strengthen the output-level consistency framing. The yes/no task should be explicitly described as forced-choice behavioral evidence, not full logical reasoning.
- Distinguish grammatical role reversal, semantic plausibility, and selectional-preference artifacts.
- Add logical consistency, behavioral testing, NLI, argument-role, and negation-related citations.

## Should Fix

- Make the output-level experiment more central for the workshop audience.
- Add exact prompts and output parsing details.
- Add confidence intervals for output accuracies.
- Use qualitative error examples to make failures concrete.

## Acceptance Risk

High before revision. The work reads as psycholinguistic probing unless the logical object, consistency criterion, and behavioral bridge are made explicit.
